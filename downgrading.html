    HiyaCFW (Downgrading)

    If you need help, ask the Nintendo Homebrew Discord!.
    I worked hard on this guide! If you want, you can donate through PayPal or Bitcoin!

    Now we will be downgrading from DSi version 1.4.5 to version 1.4

    You MUST have a compatible DSiWare game/application installed in order for this to work.

    Doing anything incorrectly MAY result in a brick. Be careful and follow the guide EXACTLY to prevent bricking.

    What you need

    • The latest release of Flipnotehax
    • The latest release of DSiBrowserhax
    • The latest release of 4Swordshax
    • The latest release of Sudokuhax
    • The latest release of FWtool
    • The latest release of the DSi Homebrew package
